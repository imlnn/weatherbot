# weather_bot
@GetFastWeatherBot

# Install dependencies
```bash
pip install -r requirements.txt
```
# Start bot
```bash
cd weatherbot
python3 main.py
```
# Run integration test
```bash
cd tests
python3 integration_test.py
```
# Run unit tests
```bash
cd tests
python3 /"your_path_to_pytest_runner"/_jb_pytest_runner.py --path test_units.py
```
